Options is required field

```js
<Select
  options={[
    { text: 'One', value: 1, defaultSelected: true },
    { text: 'Two', value: 2 },
    { text: 'Three', value: 3 }
  ]}
/>
```

Form can be readOnly

```js
<Select
  readOnly
  options={[
    { text: 'One', value: 1, defaultSelected: true },
    { text: 'Two', value: 2 },
    { text: 'Three', value: 3 }
  ]}
/>
```
