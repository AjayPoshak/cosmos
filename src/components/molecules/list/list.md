`import List from 'cosmos/list'`

---

Use this component to layout a list of `components` vertically

```js props
<List>
  <div>one</div>
  <div>two</div>
  <div>three</div>
</List>
```

#### Examples

```js
<List>
  <Stack>
    <div>github</div> <div>GitHub</div> <Switch on />
  </Stack>
  <Stack>
    <div>google-oauth2</div> <div>Google</div> <Switch off />
  </Stack>
</List>
```
